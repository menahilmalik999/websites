# Smell — Luxury Perfume Website

A modern and elegant **perfume website** built using **HTML & CSS**, designed to showcase luxury fragrances with a visually appealing and responsive layout.

---

##  Features

-  Beautiful **Hero Section** with branding and call-to-action  
-  **Top-Selling Perfume Collection** showcase  
-  **Product Filtering System** (All, New, Best Selling)  
-  Attractive **Discount & Promotion Section**  
-  **Fragrance Categories** (Floral, Woody, Fresh)  
-  **Men Collection**  
-  **Women Collection**  
-  **Community/Social Gallery Section**  
-  **Newsletter Subscription UI**  
-  Fully **Responsive Design** (Mobile + Desktop)

---

##  Technologies Used

- **HTML5**  
- **CSS3**  
- Responsive Design (Media Queries)
